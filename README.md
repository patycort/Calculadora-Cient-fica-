# Calculadora-Cient-fica-
El siguiente programa resuelve diferentes tipos de  operaciones aritméticas, geométricas(de triángulos, rectángulos y círculos), matriciales (con matrices 3x3), algebraicas y vectoriales (con vectores de dos dimensiones). Por lo que el código esta conformado por diferentes clases ligadas en las que se van heredando las funciones necesarias para la clase siguiente.
